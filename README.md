# Breast Cancer Prediction using Machine Learning
*A Machine Learning-based Breast Cancer Recognition System developed using Python and Scikit-learn to classify tumors as Malignant or Benign based on diagnostic medical features. The project includes complete data preprocessing, exploratory data analysis, visualization, model training, evaluation, and prediction using Logistic Regression.*

## 🩺 Breast Cancer Recognition System using Machine Learning

📌 Project Overview

The Breast Cancer Recognition System is a Machine Learning project developed to classify breast tumors as Malignant or Benign using diagnostic medical data.

This project demonstrates the complete Machine Learning workflow including:

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Model Building
- Model Evaluation
-Prediction System

The project was built using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

🎯 Objective

The main objective of this project is to build an efficient Machine Learning model capable of assisting in early breast cancer detection by analyzing medical diagnostic features.

📂 Dataset Information

The dataset contains various medical attributes related to breast cancer diagnosis such as:

- Radius
- Texture
- Perimeter
- Area
- Smoothness
- Compactness
- Concavity
- Symmetry
- Fractal Dimension

**Target Variable:**
- M → Malignant
- B → Benign
